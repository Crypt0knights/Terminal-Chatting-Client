# Chat-Bash ✨
## A Terminal-Chatting-Client👨‍💻👩‍💻

A multi-user chatting client implemented using Sockets in C Programming Language.

## Setup

Server-side:<br>
`gcc server.c`<br>
`./a.out <port>`
 
 Client-side:<br>
`gcc client.c`<br>
`./a.out <ip> <port>`

## Major OS Libraries Used:
* `sys/socket.h`
* `sys/types.h`
* `pthread.h`
* `netinet/in.h`
* `arpa/inet.h`
